Semester Transition Advising Tool(STAT)

Project Summary

The Semester Transition Advising Tool also known as STAT was a project created by me and 4 others as part of MSOE's software development laboratory. This project was created to help MSOE advisors plan an appropriate schedule for students who will be transitioning from the school's quarter based system to its new semester based system. It does this by accounting for the semester equivalents for all the quarter courses taken so far and then does calculations to generate an appropriate plan involving the required semester courses needed to graduate by their senior year. This project was worked on for 6 months to a point of near completion and was then passed on to MSOE professors to wrap it up over the summer.  

Project Analysis

This project was a very fun and intense experience. Since we started this project partway into the course we only had around 6 months to get it into a working state. In that time we managed to write an SRS document for the project, determine the technologies we would use, create a development environment, and develop all the core features of the project. We decided to use HTML, CSS, and javascript with react.js. Course scheduling was definitely a complex thing to automate since there were a lot of edge cases to consider(transferring students, failing courses, etc.) but we managed to get everything that was expected and more done during that time due to good communication with the PO and our clients. This project was constantly evolving as we went since our requirements would change as we got more information on how the MSOE semester transition would work since that was also a work in progress at the time. Some of the features I worked on included importing and scraping of transcript pdfs, calculating current credits and credit requirements, course plan exporting, and general defect fixes.
